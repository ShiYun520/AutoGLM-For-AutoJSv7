# AutoJS 智能语音助手 - 项目介绍

这是一款基于 AutoJS V7开发的智能手机自动化助手，融合了讯飞语音识别与 AI 大模型技术。

用户可通过底部弹窗选择文字输入或语音输入两种交互方式，用自然语言描述需求，如"打开微信给张三发消息"。系统会实时录音、调用讯飞 WebAPI 进行语音转文字，再由 AI 模型理解指令并自动执行屏幕操作。

核心架构采用 PhoneAgent 代理模式，支持多步骤任务执行、敏感操作确认、人工介入接管等安全机制。界面设计模仿输入法风格，从底部平滑弹出，支持动画效果，体验流畅自然。

执行完成后自动弹出"继续/退出"选择，实现连续对话式操作，让手机自动化变得简单高效。
